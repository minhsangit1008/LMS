# LMS API (Demo)

This repository provides a small FastAPI service and analytics notebook for KPI demos.

## Prerequisites
- Python 3.11+
- pip

## 1) Generate demo data

The API and notebooks read CSVs generated by the demo generator.

```powershell
cd C:\xampp\htdocs\moodle\LMS
python .\analytics\gen_demo_for_analytics.py
```

This will populate:
```
LMS\db\sample_data\*.csv
```

## 2) Install API dependencies

```powershell
cd C:\xampp\htdocs\moodle\LMS
python -m pip install -r .\API\requirements.txt
```

## 3) Start API

```powershell
cd C:\xampp\htdocs\moodle\LMS
uvicorn LMS.API.main:app --host 0.0.0.0 --port 8000
```

## 4) Quick checks

```powershell
curl http://127.0.0.1:8000/health
curl "http://127.0.0.1:8000/student/1/dashboard?course_id=1"
curl "http://127.0.0.1:8000/teacher/course/1/dashboard"
curl "http://127.0.0.1:8000/teacher/1/dashboard"
```

## Notes

- The demo generator is synthetic data and is not a full Moodle schema.
- CSV data is intentionally excluded from git; regenerate locally as needed.
